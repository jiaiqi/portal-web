# 内容管理系统用户操作手册

## 系统概述

本内容管理系统（CMS）由三个部分组成：

- **门户端** - 面向访客的前端展示网站
- **后台管理** - 管理员进行内容管理的操作平台
- **后端服务** - 提供数据接口和业务逻辑支持

---

## 一、系统访问

### 1.1 访问地址

| 系统     | 访问地址                      | 说明                     |
| -------- | ----------------------------- | ------------------------ |
| 门户端   | http://your-domain.com/       | 访客访问的前台网站       |
| 后台管理 | http://your-domain.com/admin/ | 管理员操作平台           |
| 后端API  | http://your-domain.com/api/   | 数据接口（无需直接访问） |

### 1.2 登录后台管理

1. 打开浏览器，访问 `http://your-domain.com/admin/`
2. 输入账号密码：
   - 默认管理员账号：`admin`
   - 默认密码：`admin123`
3. 如验证码已开启，输入验证码
4. 点击"登录"按钮进入系统

## 二、门户端功能介绍

## 三、后台管理功能详解

### 3.1 工作台

登录后默认进入工作台页面，包含：

- **数据统计** - 文章数量、访问量等核心指标
- **快捷入口** - 常用功能的快速链接
- **最近操作** - 近期的系统操作记录

### 3.2 内容管理

#### 3.2.1 文章管理

**功能路径**：内容管理 > 文章管理

**主要操作**：

1. **发布文章**

   - 点击"新增"按钮
   - 填写文章标题、分类、内容
   - 上传封面图片（可选）
   - 设置发布状态：草稿/已发布
   - 点击"提交"
2. **编辑文章**

   - 在文章列表中找到目标文章
   - 点击"编辑"按钮
   - 修改内容后点击"提交"
3. **删除文章**

   - 在文章列表中选择文章
   - 点击"删除"按钮
   - 确认删除操作
4. **文章审核**

   - 查看待审核文章列表
   - 点击"审核"查看详情
   - 选择"通过"或"拒绝"

#### 3.2.2 分类管理

**功能路径**：内容管理 > 分类管理

**主要操作**：

- 创建文章分类
- 设置分类层级（一级/二级分类）
- 调整分类排序
- 启用/禁用分类

#### 3.2.3 轮播图管理

**功能路径**：内容管理 > 轮播图管理

**主要操作**：

- 上传轮播图片
- 设置跳转链接
- 调整显示顺序
- 设置显示位置（首页/焦点图）

### 3.3 系统管理

#### 3.3.1 用户管理

**功能路径**：系统管理 > 用户管理

**主要操作**：

1. **添加用户**

   - 点击"新增"
   - 填写用户名、昵称、密码
   - 分配角色和部门
   - 设置状态（启用/禁用）
2. **分配角色**

   - 选择用户
   - 点击"编辑"
   - 在角色选项中选择对应角色
   - 保存修改

#### 3.3.2 角色管理

**功能路径**：系统管理 > 角色管理

**主要操作**：

- 创建新角色
- 分配菜单权限
- 设置数据权限范围
- 关联用户

**内置角色**：

- **超级管理员** - 拥有所有权限
- **普通角色** - 基础操作权限
- **内容审核员** - 文章审核权限

#### 3.3.3 菜单管理

**功能路径**：系统管理 > 菜单管理

**主要操作**：

- 配置系统菜单结构
- 设置菜单图标和排序
- 配置权限标识
- 启用/禁用菜单

### 3.4 系统监控

#### 3.4.1 在线用户

**功能路径**：系统监控 > 在线用户

查看当前登录系统的用户：

- 用户账号和登录时间
- 登录IP地址
- 强制下线操作

#### 3.4.2 操作日志

**功能路径**：系统监控 > 操作日志

查看系统操作记录：

- 操作人员和时间
- 操作类型（新增/修改/删除）
- 操作结果（成功/失败）
- 请求参数和响应数据

#### 3.4.3 登录日志

**功能路径**：系统监控 > 登录日志

查看用户登录记录：

- 登录账号和时间
- 登录IP和地点
- 登录结果（成功/失败）
- 浏览器信息

### 3.5 系统工具

#### 3.5.1 代码生成

**功能路径**：系统工具 > 代码生成

**使用步骤**：

1. 导入数据库表
2. 配置生成信息（包名、模块名等）
3. 预览生成代码
4. 下载或复制代码

#### 3.5.2 系统接口

**功能路径**：系统工具 > 系统接口

查看系统API文档：

- 接口列表和说明
- 请求参数和响应示例
- 在线调试接口

## 四、常用操作流程

### 4.1 发布一篇文章

1. 登录后台管理
2. 进入"内容管理"  选择要增加文章的栏目，栏目跟门户网站的导航是一一对应的。
3. 点击"新增"按钮
4. 填写文章信息：
   - 标题：文章标题
   - 分类：对应栏目的分类会自动选择
   - 内容：使用富文本编辑器编辑或者选择内容来源为外部链接，然后输入要打开的外部链接
   - 封面：上传封面图片
   - 状态：选择"已发布" 或者在内容审核菜单批量发布
5. 点击"提交"
6. 在门户端查看发布效果

### 4.2 添加新用户

1. 进入"系统管理" > "用户管理"
2. 点击"新增"
3. 填写用户信息：
   - 用户名：登录账号
   - 昵称：显示名称
   - 密码：初始密码
   - 角色：分配角色权限
4. 点击"确定"
5. 通知用户使用账号密码登录

## 五、注意事项

### 5.1 安全建议

1. **定期修改密码** - 建议每3个月修改一次密码
2. **密码复杂度** - 使用包含字母、数字、特殊字符的复杂密码
3. **退出登录** - 离开电脑前点击右上角头像选择"退出登录"
4. **权限分配** - 按最小权限原则分配用户权限

### 5.2 内容规范

1. **文章标题** - 简洁明了，不超过50个字符
2. **文章内容** - 图文并茂，排版整洁
3. **图片尺寸** - 封面图建议 800x450 像素
4. **敏感信息** - 不要发布涉及隐私或敏感的内容

### 5.3 系统维护

1. **定期备份** - 建议每周备份数据库
2. **清理日志** - 定期清理过期的操作日志
3. **监控状态** - 关注系统监控中的异常信息

## 六、常见问题

### Q1: 忘记密码怎么办？

A: 联系系统管理员重置密码。管理员可在"系统管理" > "用户管理"中重置用户密码。

### Q2: 上传图片失败？

A: 检查图片大小是否超过限制（默认50MB），格式是否支持（jpg/png/gif）。

### Q3: 文章发布后前台不显示？

A: 检查文章状态是否为"已发布"，分类是否启用，发布时间是否已到。

### Q4: 如何修改网站标题和Logo？

A: 进入"系统管理" > "参数设置"，修改站点相关配置参数。

### Q5: 系统响应慢？

A: 检查网络连接，清除浏览器缓存，或联系技术人员检查服务器状态。
